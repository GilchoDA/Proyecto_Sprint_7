# Proyecto_Sprint_7

Esta aplicación web (Streamlit) permite explorar un conjunto de datos de anuncios de venta de coches (vehicles_us.csv).
Incluye visualizaciones interactivas construidas con Plotly Express.

## Funcionalidad
- Botón para construir un histograma (kilometraje / odometer).
- Botón para construir un gráfico de dispersión (kilometraje vs precio).
- Vista previa del dataset para validación rápida.

## Tecnologías
- pandas
- streamlit
- plotly-express
